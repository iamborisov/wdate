WDate
===

Installation
---

1. Install composer: [https://getcomposer.org/download/]

2. Clone this repo:
    ```
    git clone https://github.com/iamborisov/wdate.git
    cd wdate
    ```

3. Download dependencies:
    ```
    composer update
    
    ```
4. Launch PHPUnit tests:
    ```
    ./test
    ```
